# To-Do App React

A simple to-do list application built using React.js to learn the basics of State and Props.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed (Done/Not Done)
- Localstorage

## How to Run Locally

1. Clone this repository
2. Enter the project directory: `cd project-name`
3. Install dependencies: `npm install`
4. Run the application: `npm start`

## Technologies Used

- React.js
- Tailwind
